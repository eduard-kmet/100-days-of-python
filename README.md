# 100 Days of Python 🐍

My personal learning journey through the **100 Days of Code: Python Pro Bootcamp** — documented day by day.

## 🎯 Why I'm learning Python

I work with **Revit** and BIM tools daily. Most of the repetitive work — generating reports, extracting model data, processing parameters — can be automated. Python is the key to making that happen.

Long-term goals:
- Automate BIM data extraction from Revit models using the Revit API + Python
- Build scripts that replace manual, error-prone tasks in construction projects
- Connect BIM data to reporting tools (Power BI, Excel) without manual exports
